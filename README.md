# 敏感词撤回插件
监测群聊的发言合规，管理员权限可阶梯式禁言，下载即用  
支持白名单控制 支持配置管理员私信提醒 支持自定义敏感词 支持API远程检测 支持免处罚用户  

# 主要功能
- **自动监控** - 自动检测群消息中的敏感词  
- **智能检测** - 支持本地检查和远程API检查双引擎  
- **自动处理** - 自动撤回违规消息、自动禁言违规用户  
- **分级禁言** - 根据违规次数自动提级禁言（60秒→10分钟→1天）  
- **免处罚机制** - 群管理员/群主自动免处罚，同时支持配置指定QQ号免处罚  
- **管理员私信提醒** - 检测到违规时通过私聊通知管理员，支持自定义模板  
- **数据库记录** - 记录所有违规信息，支持数据分析  
- **统计分析** - 详细的违规统计和数据报告  

# 配置项
| 配置项 | 说明 |
|--------|------|
| group_whitelist | 监控的白名单群号列表（直接填写QQ群号） |
| admin_qq_list | 接收提醒的管理员QQ号列表（直接填写QQ号） |
| api_endpoint_enabled | 是否开启API检测功能 |
| api_endpoint | 敏感词检测API地址 |
| group_notice_enabled | 是否在群内发送敏感词提醒 |
| notice_template | 群内提醒消息模板 |
| admin_notice_template | 管理员私聊提醒模板 |
| enable_auto_ban | 是否启用自动禁言 |
| enable_message_delete | 是否启用自动撤回消息 |
| enable_local_check | 是否启用本地关键词检测 |
| exempt_users | 免处罚用户QQ号列表（群管理员/群主自动免处罚，无需配置） |
| custom_forbidden_words | 自定义违禁词列表 |
| custom_white_words | 自定义白名单词列表 |
| ban_rules | 阶梯式禁言规则（首次/二次/三次禁言时长、重置时间） |
| cooldown_seconds | 同一用户检测冷却时间（秒） |
| max_log_days | 违规记录保留天数 |
| statistics_enabled | 是否启用统计功能 |
| bypass_rate_limit | 是否绕过限流策略 |
| debug_mode | 是否启用调试模式 |

# 指令列表
| 指令 | 说明 | 权限 |
|------|------|------|
| 敏感词统计 | 查看检测统计数据 | 管理员 |
| 违规记录 | 查看违规记录 | 管理员 |
| 重置违规记录 | 重置违规记录 | 管理员 |
| 测试违规 | 发送测试违规通知 | 管理员 |
| 敏感词测试 | 测试文本是否包含敏感词 | 管理员 |
| 添加敏感词 | 添加自定义敏感词 | 管理员 |
| 删除敏感词 | 删除自定义敏感词 | 管理员 |
| 敏感词列表 | 查看自定义敏感词列表 | 管理员 |
| 敏感词撤回插件状态 | 查看插件运行状态 | 所有人 |
